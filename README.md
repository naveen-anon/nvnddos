## ⚡ About

**NVNDDOS** is a command-line tool designed for network load testing, packet generation, and traffic simulation in controlled environments.

It can be used for:

- 🛡️ Security lab testing  
- 📡 Network stress analysis  
- 🧪 Educational demonstrations  
- ⚙️ Performance benchmarking  

> ⚠️ Use only on systems you own or have written permission to test.

---

## ✨ Features

- 🚀 High-speed packet sending  
- 🧵 Multi-threaded performance  
- 🌐 Supports IP / Host targeting  
- ⚡ Lightweight and fast  
- 🖥️ Linux / Kali / Termux compatible  
- 📊 Real-time console output

## 📦 Installation

### Kali Linux / Ubuntu

```bash
git clone https://github.com/naveen-anon/nvnddos.git
cd nvnddos
pip3 install -r requirements.txt
python3 nvnddos.py
```
## termux installation 
```bash
pkg update && pkg upgrade
pkg install git python -y
git clone https://github.com/naveen-anon/nvnddos.git
cd nvnddos
pip install -r requirements.txt
python nvnddos.py
```
